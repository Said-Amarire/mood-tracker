# Mood Tracker App

## Project Description
This is a Mood Tracker application that allows users to record and monitor their daily moods. The project is built using HTML, CSS, JavaScript, and optionally React with Tailwind CSS.

## Features
- Record daily mood
- View mood history
- Responsive UI for desktop and mobile

## Technologies Used
- HTML
- CSS / Tailwind CSS
- JavaScript / React
- Local Storage for saving user data
